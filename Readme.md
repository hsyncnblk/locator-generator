This browser extension does not collect, store, transmit, or process any personal data from users.
All actions performed by the extension occur locally within the user’s browser.

Data Collection

The extension does not collect any of the following:

Personal information (name, email, address, etc.)

Financial or payment data

Health information

Authentication or login data

Device or location data

Browsing history

User activity (clicks, scrolls, keystrokes, etc.)

Website content (text, images, media, etc.)

Data Sharing

No data is shared with any third party, service, or external server because no data is collected.

Permissions

The extension only uses browser permissions necessary to function, such as reading the current tab’s HTML to identify and highlight seller information.
No data from this process is stored or transmitted.
